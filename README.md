# 实现的功能
1. 查看上课时间、地点、教师和课程时间
2. 查看学校政策和特殊条款
3. 中英文两种版本(在右上角点击切换)
4. 登录界面(左上角有注销和切换账户)
5. 可在搜索栏直接搜索问题
    1. 根据搜索栏的关键字搜索相关的问题和答案
    2. 如果没有查到，则下面两种方式
        * 询问用户并显示一些受欢迎的问题，选择提供的问题并获取答案
        * 选择"我没有找到我要问的问题"，网页跳到下一个界面，然后上传自己的问题
    3. 搜索栏下面有热门问题和解答
6. 网页的底部有学校各个部门的电话号码和电子邮件地址，可以选择手动联系
7. 学生和老师可以在网页上找到课表：包括时间和地点等信息
8. 学生和老师可以在网站上查看学校政策和相关规定，并且可以搜索
