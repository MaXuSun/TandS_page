# Frontend
## Requirement
```
node.js==12.18.2
vue-cli==4.1.1
vue==2.6.1
```

## Function
- Sign in and Sign up
- Choose system language
- Get schedule
- Switch account and Log out 
- Search for question
- Fuzzy search
- Upload a question
- Answer a question
- Check school policy and related regulations
- Get the contact information of the school
- Show hot topics


## Project setup
```
npm install
```
